Working Copy for students

Working Copy is a powerful iOS client for iPhone and iPad. All pro features are available to students for free as part of the GitHub Student Developer Pack.

Visit this page from a iPhone or iPad where Working Copy is installed. Inside the app you will need to sign into your GitHub account.

If this account is registered to a student developer you will be able to access all pro features as long as you are logged into this GitHub account.

*** Edit your projects right from your mobile device, pushing back changes to GitHub, create pull requests, deploy to servers. ***

Read the manual or the newsletter and get in touch by email or on Twitter.
